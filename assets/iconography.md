# 沐研萃 · 圖標使用

## 系統
- **Lucide Icons** · 1.5px stroke · 灰色描邊
- CDN: `https://unpkg.com/lucide@latest` 或 inline SVG

## 常用
| 用途 | 圖標 |
|---|---|
| 檢驗/信任 | `shield-check` |
| 醫療/營養師 | `stethoscope`, `heart-pulse` |
| 適用族群 | `user-check` |
| 使用提醒 | `info` |
| 配送 | `truck` |
| 購物車 | `shopping-bag` |
| 搜尋 | `search` |
| 會員 | `user` |

## 規範
- 預設尺寸 20 / 24 / 32px
- 預設色 `var(--fg-3)` (#7a7565)
- 強調色用主色 `var(--sage-500)` 或重音 `var(--sea-500)`
- **禁用**：emoji、font-awesome 彩色、填滿式圖標（違反日系極簡）
